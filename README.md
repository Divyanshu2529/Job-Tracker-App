# Job-Tracker-App
A full-stack Job Tracker application that allows users to manage and track their job applications. Users can create, view, update, delete, and filter job applications through a clean and simple interface.


## Features

### Job Application CRUD
Users can:
- Add a job application with:
  - Company name
  - Job title
  - Application status (Applied, Interviewing, Rejected, Offer)
  - Application date
  - Notes
- View all job applications
- Update existing job applications
- Delete job applications

### Filtering
- Filter job applications by status

### Backend API
- REST API built with Node.js and Express
- API Routes:
  - `GET /jobs`
  - `POST /jobs`
  - `PUT /jobs/:id`
  - `DELETE /jobs/:id`

### Frontend
- Built with React
- Simple and clean UI
- Loading and error states for better user experience
